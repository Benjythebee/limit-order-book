# limit-order-book

A JavaScript limit order book implementation supporting limit and market orders.

## Install
```
$ npm install limit-order-book
```

## Testing
```
$ git clone https://github.com/rhodey/limit-order-book
$ cd limit-order-book
$ npm install
$ npm run test
```
